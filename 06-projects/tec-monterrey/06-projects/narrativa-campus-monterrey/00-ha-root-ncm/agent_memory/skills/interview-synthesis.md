# Skill: Interview Synthesis

**Trigger:** `\synthesize [source]` or "synthesize interviews from [category]"

## Purpose
Synthesize insights from interview transcripts into coherent themes that inform the narrative.

## Protocol

1. **IDENTIFY SOURCE**
   - Leadership: `02-community-ncm/entrevistas/liderazgo-transcripciones/`
   - Directors: `02-community-ncm/entrevistas/directivos/`
   - Students: `02-community-ncm/entrevistas/estudiantes/`

2. **READ TRANSCRIPTS**
   - Note key quotes (verbatim when powerful)
   - Identify recurring themes
   - Flag tensions or contradictions
   - Mark unique insights

3. **CLUSTER THEMES**
   - Group by narrative section
   - Group by stakeholder perspective
   - Group by 3 bets relevance

4. **SYNTHESIZE**
   - Create summary of cross-cutting themes
   - Preserve individual voice where distinctive
   - Note consensus vs divergence
   - Highlight actionable insights

5. **OUTPUT**
   - Summary document with:
     - Theme clusters
     - Key quotes (attributed)
     - Tensions surfaced
     - Recommendations for narrative
