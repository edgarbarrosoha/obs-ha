# Legacy — Horizons Architecture Core

> El propósito, la visión y los fundamentos teóricos del framework HA.

## Visión de HA

> Convertirse en una plataforma global transformadora para inteligencia híbrida, combinando IA, experiencia humana y un framework de pensamiento sistémico.

## Misión

Crear un framework que permita a individuos e instituciones navegar complejidad a través de la coordinación adaptativa humano-IA.
