# HA Root — SIA-DOCET

> Sistema de Inteligencia Aumentada para DOCET: Una escuela K-9 AI-First

## Visión del Proyecto

Desarrollar el sistema nervioso digital de DOCET que otorgue **omnisciencia operativa**, **fair accountability** y **tiempo de calidad** a través de una arquitectura multi-agente que libere a directivos y docentes de carga administrativa para enfocarse en la experiencia vivencial del alumno.

## Concepto Central: La Ancilla

El sistema está diseñado como una "Ancilla" (concepto Cortana/Halo): agentes que **empujan (nudge)** al directivo hacia sus metas, recordándole prioridades y detectando desviaciones automáticamente.

```
Usuario (Directivo)
        │
        ▼
  ┌─────────────┐
  │  HA RAÍZ    │ ← Un solo chat
  │ (Copiloto)  │
  └──────┬──────┘
         │
    ┌────┴────┐
    ▼         ▼
┌───────┐ ┌───────┐ ... (6 agentes)
│Legacy │ │Commun.│
└───────┘ └───────┘
```

> Un chat sencillo hacia afuera, una red agéntica poderosa hacia adentro.
