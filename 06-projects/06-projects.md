## Proceso de Gestión

### Ciclo de Vida

```
Oportunidad ──▶ Propuesta ──▶ Kick-off ──▶ Ejecución ──▶ Cierre
     │              │            │            │           │
     ▼              ▼            ▼            ▼           ▼
  Evaluar      Negociar      Planear      Entregar   Documentar
```

### Governance

| Actividad | Frecuencia | Participantes |
|-----------|------------|---------------|
| Review de portafolio | Semanal | Equipo core |
| Status de proyecto | Semanal | Por proyecto |
| Retrospectiva | Al cierre | Equipo proyecto |

## Métricas de Portafolio

| Métrica | Meta | Actual |
|---------|------|--------|
| Proyectos activos | 3-5 | 5 |
| Tasa de conversión propuestas | > 50% | - |
| Satisfacción cliente | > 8/10 | - |
| Contribución a [[01-legacy\|legado]] | 100% | - |

## Evita
- Actividad sin impacto
- Proyectos sin aprendizaje
- Desalineación con el [[01-legacy|legado]]
- Sobrecarga del equipo
