**1. Contextualización**

- Se recopilaron y analizaron datos socioeconómicos, indicadores clave y programas de gobierno anteriores.
- Se evaluó la trazabilidad y se extrajeron aprendizajes de programas de gobierno previos.
- Se analizó la alineación de las propuestas de campaña con el Plan Estatal de Desarrollo (PED) 2050 (generando documentos y grafos).
- Se realizó un diagnóstico preliminar basado en hallazgos documentales y estadísticos.
- Se definió una estructura temática inicial para el Programa de Gobierno.
- Se realizó un análisis de tendencias nacionales e internacionales mediante IA (reubicado de la planificación inicial).

**2. Arranque Formal**

- Se realizaron entrevistas con la Gobernadora y actores clave para definir visión, misión, prioridades y proyectos emblemáticos.
- Se analizó la información de las entrevistas para extraer la visión de la Gobernadora.
- Se llevaron a cabo mesas de trabajo iniciales con coordinadores de ejes y titulares de dependencias.
- Se diseñó y distribuyó un formato para que las dependencias integraran sus propuestas iniciales.
- Se recopilaron y organizaron las propuestas iniciales de las dependencias en una base de datos.

**3. Consulta Social**

- Se diseñó la metodología general para la Consulta Social, incluyendo diversos canales de participación.
- Se elaboraron y validaron cuestionarios específicos para cada modalidad de consulta.
- Se ejecutaron los diferentes mecanismos de consulta:
    - Mesas de trabajo por eje con actores clave y expertos.
    - Entrevistas con titulares.
    - Talleres regionales por grupos de población.
    - Consultas en línea (comunidad escolar, redes sociales).
    - Sesiones en centros Nuevo Comienzo.
- Se documentaron los resultados individuales de cada mecanismo de consulta implementado.
- Se procesaron y analizaron los resultados globales de la consulta, utilizando Procesamiento de Lenguaje Natural (PLN) / IA para extraer ideas clave y percepciones públicas.
- Se generó un reporte general sintetizado con los hallazgos y temas priorizados de la Consulta Social.
- Se procesó y analizó la información de las entrevistas y mesas con el funcionariado (realizadas en el arranque) para generar conclusiones y una base de datos de prioridades estatales.
- Se presentaron resultados preliminares a la Gobernadora.
- Se realizó el cierre formal de la consulta.
- Se conformó el Consejo de Planeación para el Desarrollo del Estado de Guanajuato (COPLADEG).

**4. Integración del Planteamiento Estratégico**

- Se consolidó la información de la consulta social y las prioridades gubernamentales en una base de datos central.
- Se aplicaron herramientas de IA para procesar los grandes volúmenes de datos, identificar patrones y asistir en la priorización de objetivos, estrategias y acciones.
- Se generó un documento de análisis estratégico con recomendaciones concretas, alineando hallazgos con prioridades estatales, demandas ciudadanas y el PED 2050.
- Se definió la metodología para estructurar los objetivos, estrategias, programas y acciones del Programa de Gobierno.
- Se elaboró la propuesta del planteamiento estratégico (contenido del Programa).
- Se diseñaron, en conjunto con las dependencias, los indicadores de desempeño y resultado.
- Se desarrolló y utilizó una herramienta (con IA) para analizar trayectorias históricas de indicadores y facilitar la definición de metas anualizadas.
- Se desarrolló y utilizó un formulario/plataforma para recoger los metadatos de los indicadores (Fichas Técnicas de Indicadores - FTI).
- Se recopiló la información de las FTI en una base de datos.
- Se generaron las FTI en formato editable.
- Se prepararon materiales y se co-diseñó la metodología para las reuniones de validación del planteamiento estratégico con las dependencias.
- Se realizaron sesiones de validación institucional por pares (dependencias revisando y avalando contenido).
- El COPLADEG validó el contenido del programa.
- Se integró una versión síntesis del Programa de Gobierno (máx. 200 páginas).
- Se desarrolló "Esperanza", el agente de IA (chatbot) para facilitar la búsqueda de contenido dentro del Programa.

**5. Integración y Difusión**

- Se elaboró la versión amplia y final del Programa de Gobierno 2024-2030 en formato editable.
- Se diseñó y elaboró una versión ejecutiva tipo infografía (máx. 30 páginas) con guías de impresión.
- Se desarrolló una presentación ejecutiva en formato PPTX.
- Se elaboraron versiones adaptadas del Programa para niñas, niños, personas con discapacidad y pueblos indígenas.
- Se desarrolló y puso en línea el sitio web del proyecto ([guanajuatointeligente.com](http://guanajuatointeligente.com/)) para alojar información del proceso y documentos finales.
- Se produjeron tres videos informativos sobre el Programa y la consulta social.**
