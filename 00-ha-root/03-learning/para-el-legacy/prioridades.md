# Prioridades de Aprendizaje

> ¿Qué aprender primero para lograr el legacy?

## Framework de Priorización

```
Prioridad = (Impacto en Legacy × Urgencia) / Esfuerzo
```

| Factor | Pregunta |
|--------|----------|
| Impacto | ¿Cuánto acerca al legacy? (1-5) |
| Urgencia | ¿Hay deadline o dependencia? (1-5) |
| Esfuerzo | ¿Cuánto tiempo/recursos? (1-5, menor=mejor) |
