# Cola de Tareas Automatizadas

## Instrucciones
Cuando Edgar escriba una tarea con formato de acción, el asistente debe:
1. Identificar el tipo de tarea
2. Ejecutarla o prepararla
3. Presentar el resultado para aprobación
4. Si se aprueba, marcar como completada

## Tipos de Tareas Soportadas

### Correos
**Trigger**: "escribir correo a [persona] sobre/diciendo [tema]"
**Output**: Correo completo listo para copiar

### Follow-ups
**Trigger**: "hacer follow-up a [persona/proyecto]"
**Output**: Correo de seguimiento basado en contexto del proyecto

### Resúmenes
**Trigger**: "resumir [documento/reunión/tema]"
**Output**: Resumen estructurado con puntos clave

### Agendas
**Trigger**: "preparar agenda para [reunión]"
**Output**: Agenda con temas, tiempos sugeridos, participantes

### Reportes
**Trigger**: "hacer reporte de [proyecto] para [audiencia]"
**Output**: Reporte estructurado según el destinatario
