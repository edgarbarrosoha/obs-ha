## Cómo Usar Este Sistema

### Para ver el grafo
1. Abre **Graph View** en Obsidian (`Ctrl/Cmd + G`)
2. El grafo mostrará las conexiones entre dimensiones
3. Los colores representan cada dimensión

### Para navegar
1. Empieza en este archivo (HA-Central)
2. Navega a la dimensión relevante
3. Explora los documentos dentro de cada dimensión

### Para contribuir
1. Identifica la dimensión correcta
2. Crea o edita el documento
3. Agrega [[wikilinks]] a documentos relacionados
4. El grafo se actualiza automáticamente

## Principios del Sistema

1. **Todo conectado** — Usa [[wikilinks]] siempre
2. **Una fuente de verdad** — No duplicar información
3. **Actualización continua** — Documentos vivos
4. **Dimensión correcta** — Cada cosa en su lugar
