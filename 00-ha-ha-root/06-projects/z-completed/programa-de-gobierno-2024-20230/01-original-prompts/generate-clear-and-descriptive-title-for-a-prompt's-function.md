**Context:**  
You are tasked with generating a **descriptive and clear title** for a given prompt. The title should accurately reflect what the prompt **does** and its **intended function**, making it easy to understand at a glance.

---

**Instructions:**

1. **Input Data:**
    
    - **Original Prompt**: Insert the original prompt text here.
2. **Task:**
    
    - **Analyze the Original Prompt**: Carefully read and understand the main function of the prompt, including its purpose and the task it performs.
    - **Generate a Descriptive Title**: Based on the function of the prompt, create a clear and descriptive title that reflects what the prompt actually does.
3. **Title Criteria**:
    
    - **Clarity**: The title should clearly communicate the main action or task that the prompt performs.
    - **Relevance**: Ensure the title is closely related to the function of the prompt and describes its purpose accurately.
    - **Conciseness**: Keep the title brief while still descriptive enough to convey the key function.
    - **Action-Oriented**: If applicable, use action verbs to indicate what the prompt helps the user accomplish (e.g., "Create," "Generate," "Organize").

---

**Output:**

- **Original Prompt**:  
    _Display the original prompt text here._
    
- **Generated Title**:  
    _Provide the new, descriptive title that reflects the function of the prompt._
