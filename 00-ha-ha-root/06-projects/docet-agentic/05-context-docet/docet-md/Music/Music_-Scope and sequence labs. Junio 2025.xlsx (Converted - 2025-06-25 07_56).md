Participates in active listening exercises, identifying tempo changes (Lento, Moderato, Allegro) and simple articulation in selected works, including national composers. |  | Integrates rhythm, melody, and dynamics in group and individual performances, applying coordination and expressiveness in vocal and instrumental execution. |  | Elementary 2 | Recognizes and plays basic rhythms with quarter, half, eighth, sixteenth, and whole notes in 2/4 and 4/4 time, using the body and percussion instruments.   Sings and listens to seconds and thirds, using Kodály hand signs and simple songs with notes from C Major and A minor.   Uses loud and soft sounds (piano, mezzoforte, forte) and different ways of singing or playing (legato and staccato) in music activities. |  | Reads and plays rhythms in 2/4, 3/4, and 4/4 meters, using instruments like percussion, keyboard, and recorder. Applies dynamics and articulation (soft/loud, smooth/detached) in short songs using voice or instruments. |  | Combines rhythm, melody, and dynamics in group or solo performances, showing more fluency in singing and playing.   Recognizes sounds and notes when listening, connecting them to songs and scales learned during the year. |  | Elementary 3 | Students will recognize and perform basic rhythmic patterns using dotted half notes, dotted quarter notes, and syncopation in 2/4 and 4/4 time through clapping and percussion activities.  Students will identify and sing short melodic phrases that include ascending and descending 2nd and 3rd intervals using their voice and melodic instruments (xylophone, piano, kalimba).  Students will describe and demonstrate different musical tempos, including Largo and Andante, using a metronome, and engage in active listening by identifying instruments and expressive elements in works by local composers. |  | Students will read and perform short melodic phrases that include 2nd, 3rd, and 4th intervals in both major (C, F) and relative minor (A, D) scales using voice and melodic instruments.  Students will demonstrate expressive musical interpretation by applying dynamics (mp, mf, crescendo, decrescendo) and articulations (staccato and legato) during individual and group performances.  Students will identify mood and tempo (Moderato and Allegro) in musical excerpts, and recognize the expressive use of rhythm and dynamics in works by Mexican composers such as Revueltas. |  | Students will perform children’s songs in ensemble settings, applying appropriate dynamics, tempo changes, and articulation, while maintaining group coordination.  Students will accurately play and sing major scales (C, F, G) and their relative minors, demonstrating understanding of pitch and tonal relationships through melodic exercises.  Students will compare and discuss musical elements such as rhythm, melody, dynamics, and instrumentation in local and global music through guided listening and reflection. |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |
|  | Elementary 4 | Students will read and perform complex rhythmic patterns including triplets, dotted notes, ties, and syncopation in simple time signatures using clapping, percussion, and notation exercises. Students will identify and interpret different tempos (Largo, Andante, Presto) and tempo changes through performance and active listening to selected musical works. |  | Students will distinguish between simple and compound meters and accurately perform rhythmic patterns in 2/4, 3/4, and 4/4 using body percussion and classroom instruments.  Students will play and sing major and minor scales containing up to two sharps or flats, demonstrating pitch accuracy and finger coordination on melodic instruments.  Students will express musical ideas using contrasting dynamics (pp–ff) and articulations (legato, staccato, accent), and will identify these expressive elements through active listening to selected composers. |  | Students will compose a 4-bar original melody using previously learned scales and intervals, applying basic principles of melodic structure.  Students will participate in ensemble performances that demonstrate coordination of rhythm, melody, dynamics, and articulation through short vocal and instrumental pieces.  Students will analyze and reflect on musical works, identifying expressive elements and connecting musical experiences to their personal and cultural identity through listening and discussion. |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |
|  | Elementary 5 | Students will read and perform rhythmic patterns in 2/4, 3/4, and 4/4 time signatures, including mixed subdivisions (such as sixteenth notes), using congas, bongos, and body percussion. Students will recognize and perform major and minor scales in C, G, D, F, and Bb, both visually and aurally, using melodic instruments and voice. Students will identify, build, and apply major chords in various keys during both compositional tasks and group performances that integrate rhythm, melody, dynamics, and articulation.  Students will use basic audio recording tools (e.g., microphones, input levels) to record and reflect on their performances, and will analyze how expression and structure relate to national musical identity through active listening. |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |

Students will perform in ensembles, integrating rhythm, melody, dynamics, and articulation to deliver coherent and expressive group performances. Students will perform with electric and acoustic instruments, integrating tempo changes, dynamics, articulation, and harmony, and will reflect on musical works by national and international composers. |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |

Students will engage in active listening to identify musical modes, harmonic progressions, form, and dynamics, with an emphasis on works by national composers and the connection between music, identity, and emotion. |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |
