|               |   sin dificultad.              |                       |
|               |                                |                       |
|               | - Al caminar, disfruta empujar |                       |

|               |                                |                       |
|               | - Sube y baja por una escalera |                       |
|               |   con ayuda, descansando ambos |                       |
|               |   pies en cada escalón.        |                       |
|               |                                |                       |
|               | - Baja las escaleras gateando  |                       |

|               |                                |                       |
|               | - De pie, puede inclinarse y   |                       |

|               |                                |                       |
|               | - Se pone en cuclillas para    |                       |
|               |   jugar.                       |                       |
|               |                                |                       |
|               | - Lanza pelotas pequeñas.      |                       |
|               |                                |                       |
|               | - Patea pelotas hacia          |                       |
|               |   adelante.                    |                       |
|               |                                |                       |
|               | - Al escuchar música, empieza  |                       |
|               |   a bailar.                    |                       |
+---------------+--------------------------------+-----------------------+
| **Motora      | - Deja caer objetos dentro de  |                       |
| Fina**        |   un recipiente pequeño.       |                       |
|               |                                |                       |
|               | - Toma el lápiz con el puño y  |                       |
|               |   con el pulgar hacia arriba.  |                       |
|               |                                |                       |
|               | - Elabora vigorosamente        |                       |
|               |   garabatos con crayolas o     |                       |
|               |   marcadores.                  |                       |
|               |                                |                       |
|               | - Apila de 4 a 6 objetos.      |                       |
|               |                                |                       |
|               | - Encaja anillos al azar en el |                       |
|               |   tablero.                     |                       |
|               |                                |                       |
|               | - Desarma juguetes diseñados   |                       |
|               |   para armar y desarmar.       |                       |
|               |                                |                       |
|               | - Levanta objetos del suelo    |                       |
|               |   con un hilo o cuerda.        |                       |
+---------------+--------------------------------+-----------------------+
| **Cognición** | - Activa un juguete después de |                       |
|               |   la demostración por un       |                       |
|               |   adulto.                      |                       |
|               |                                |                       |
|               | - Activa un juguete sin        |                       |
|               |   demostración de un adulto.   |                       |
|               |                                |                       |
|               | - Pide a un adulto que le dé   |                       |
|               |   cuerda a un juguete nuevo.   |                       |
|               |                                |                       |
|               | - Mueve carritos o juguetes    |                       |
|               |   diseñados con ruedas.        |                       |
|               |                                |                       |
|               | - Observa los dibujos de un    |                       |
|               |   cuento indicando o nombrando |                       |
|               |   objetos conocidos, si se le  |                       |
|               |   requiere.                    |                       |
|               |                                |                       |
|               | - Muestra juguetes a adultos.  |                       |
|               |                                |                       |
|               | - Encuentra objetos escondidos |                       |
|               |   (en una caja, bajo una tela  |                       |
|               |   o lienzo).                   |                       |
|               |                                |                       |
|               | - Maneja 3 o 4 juguetes,       |                       |
|               |   dejándolos de lado al        |                       |
|               |   recibir uno nuevo.           |                       |
|               |                                |                       |
|               | - Demuestra el uso de objetos  |                       |
|               |   conocidos (ejemplo: taza,    |                       |
|               |   cuchara).                    |                       |
|               |                                |                       |
|               | - Busca a una persona conocida |                       |
|               |   que ha abandonado la         |                       |
|               |   habitación.                  |                       |
|               |                                |                       |
|               | - Utiliza un objeto (palo,     |                       |
|               |   tabla) para acercar un       |                       |
|               |   juguete que está fuera de su |                       |
|               |   alcance.                     |                       |
|               |                                |                       |
|               | - Busca metódicamente un       |                       |
|               |   objeto escondido.            |                       |
+---------------+--------------------------------+-----------------------+
| **Lenguaje**  | - Expresa de manera adecuada   |                       |
|               |   de 3 a 20 palabras simples.  |                       |
|               |                                |                       |
|               | - Utiliza de 20 a 50 palabras. |                       |
|               |                                |                       |
|               | - Dice "mamá", "papá"          |                       |
|               |   discriminadamente.           |                       |
|               |                                |                       |
|               | - Dice su nombre.              |                       |
|               |                                |                       |
|               | - Comprende aproximadamente    |                       |
|               |   300 palabras.                |                       |
|               |                                |                       |
|               | - Al requerírsele, señala de 1 |                       |
|               |   a 5 partes del cuerpo.       |                       |
|               |                                |                       |
|               | - Al requerírsele, señala      |                       |
|               |   objetos conocidos.           |                       |
|               |                                |                       |
|               | - Escucha atento al nombrarle  |                       |
|               |   objetos en ilustración.      |                       |
|               |                                |                       |
|               | - Responde a preguntas simples |                       |
|               |   ("¿qué?", "¿quién?",         |                       |
|               |   "¿cuándo?").                 |                       |
|               |                                |                       |
|               | - Utiliza gestos y palabras    |                       |
|               |   rituales como "adiós",       |                       |
|               |   "hola".                      |                       |
|               |                                |                       |
|               | - Protesta diciendo "no" con   |                       |
|               |   su cabeza, retirándose o     |                       |
|               |   frunciendo el ceño.          |                       |
|               |                                |                       |
|               | - Niega y afirma con el        |                       |
|               |   movimiento de su cabeza.     |                       |
|               |                                |                       |
|               | - Utiliza gestos y             |                       |
|               |   vocalizaciones para reñir,   |                       |
|               |   bromear y avisar.            |                       |
|               |                                |                       |
|               | - Utiliza frases de una sola   |                       |
|               |   palabra para expresar un     |                       |
|               |   pensamiento completo (por    |                       |
|               |   ejemplo: "parque" por        |                       |
|               |   "llévame al parque").        |                       |
|               |                                |                       |
|               | - Combina dos palabras para    |                       |
|               |   crear frases.                |                       |
|               |                                |                       |
|               | - Elabora oraciones con un     |                       |
|               |   promedio de 2 palabras.      |                       |
|               |                                |                       |
|               | - Elabora palabras con la      |                       |
|               |   estructura                   |                       |
|               |   consonante-vocal-consonante. |                       |
|               |                                |                       |
|               | - Dice "se acabó" y pide       |                       |
|               |   "más".                       |                       |
|               |                                |                       |
|               | - Sigue instrucciones que      |                       |
|               |   indiquen "dentro", "sobre".  |                       |
|               |                                |                       |
|               | - Obedece órdenes simples (por |                       |
|               |   ejemplo: "quítate") y        |                       |
|               |   utiliza posesivos ("mío").   |                       |
+---------------+--------------------------------+-----------------------+
| **Socio**     | - Se reconoce ante un espejo   |                       |
|               |   y/o retrato.                 |                       |
| **emocional** |                                |                       |
|               | - Juega solo durante períodos  |                       |
|               |   breves.                      |                       |
|               |                                |                       |
|               | - Juega solo, aun cuando otros |                       |
|               |   niños estén presentes.       |                       |
|               |                                |                       |
|               | - Manifiesta celos.            |                       |
|               |                                |                       |
|               | - Muestra primeros indicios de |                       |
|               |   independencia al conducirse  |                       |
|               |   de manera juguetona y        |                       |
|               |   exploratoria (aunque aún     |                       |
|               |   requiere supervisión).       |                       |
|               |                                |                       |
|               | - Al conducirse de manera      |                       |
|               |   independiente, quizá         |                       |
|               |   requiera regresar a donde    |                       |
|               |   están sus papás por          |                       |
|               |   seguridad o atención         |                       |
|               |                                |                       |
|               | - Al enfrentar una situación   |                       |
|               |   nueva busca a sus padres     |                       |
|               |   para seguridad.              |                       |
|               |                                |                       |
|               | - El contacto con otros niños  |                       |
|               |   es propiciado por un adulto  |                       |
|               |   o un juguete.                |                       |
|               |                                |                       |
|               | - Es curioso, se mete en todo. |                       |
|               |                                |                       |
|               | - A menudo hace lo contrario a |                       |
|               |   lo que se le pide.           |                       |
|               |                                |                       |
|               | - Usualmente se muestra        |                       |
|               |   desafiante diciendo "no" a   |                       |
|               |   muchos requerimientos.       |                       |
|               |                                |                       |
|               | - Disfruta los juegos          |                       |
|               |   sencillos de simulación (por |                       |
|               |   ejemplo: alimentar a un      |                       |
|               |   muñeco).                     |                       |
|               |                                |                       |
|               | - Se le dificulta compartir.   |                       |
|               |                                |                       |
|               | - Ayuda a guardar las cosas.   |                       |
|               |                                |                       |
|               | - Duerme durante la noche sin  |                       |
|               |   necesidad de tomar siestas   |                       |
|               |   diurnas.                     |                       |
|               |                                |                       |
|               | - Come con cuchara y bebe en   |                       |
|               |   taza sin ayuda (aunque puede |                       |
|               |   derramar el alimento).       |                       |
|               |                                |                       |
|               | - Empieza a utilizar el        |                       |
|               |   tenedor.                     |                       |
|               |                                |                       |
|               | - Mastica la comida.           |                       |
|               |                                |                       |
|               | - Se sienta en el sanitario    |                       |
|               |   por lo menos un minuto y con |                       |
|               |   supervisión.                 |                       |
|               |                                |                       |
|               | - Indica o verbaliza sus       |                       |
|               |   necesidades fisiológicas     |                       |
|               |   aproximadamente el 50% de    |                       |
|               |   las veces.                   |                       |
|               |                                |                       |
|               | - Aparta el obstáculo que le   |                       |
|               |   impida abrir la puerta.      |                       |
|               |                                |                       |
|               | - Imita diversas acciones      |                       |
|               |   domésticas (por ejemplo:     |                       |
|               |   barrer, trapear).            |                       |
