**II. APRENDIZAJES ESPERADOS Y SECUENCIA**

+-------------------------------------+---------------------------------------------+
| **AREA**                            | **APRENDIZAJES ESPERADOS**                  |
+-------------------------------------+---------------------------------------------+
|                                                                                   |
+-------------------------------------+---------------------------------------------+
| Sensorial                           | **1. Habilidades de autoconocimiento y      |
|                                     | percepción visual**                         |
|                                     |                                             |
|                                     | - Se reconoce a sí mismo al mirar su        |
|                                     |   reflejo en el espejo.                     |
|                                     |                                             |
|                                     | **2. Habilidades auditivas y de             |
|                                     | discriminación sonora**                     |
|                                     |                                             |
|                                     | - Distingue los sonidos del ambiente y      |
|                                     |   nombra el objeto o ser vivo que los       |
|                                     |   genera.                                   |
|                                     |                                             |
|                                     | **3. Habilidades táctiles y de              |
|                                     | discriminación sensorial**                  |
|                                     |                                             |
|                                     | - Distingue a través del tacto texturas     |
|                                     |   ásperas y lisas.                          |
|                                     |                                             |
|                                     | - Distingue formas u objetos a través del   |
|                                     |   tacto y comienza a nombrarlas.            |
|                                     |                                             |
|                                     | - Distingue temperaturas a través del       |
|                                     |   tacto: frío y caliente.                   |
|                                     |                                             |
|                                     | **4. Habilidades de asociación y            |
|                                     | categorización sensorial**                  |
|                                     |                                             |
|                                     | - Puede relacionar dos objetos con la misma |
|                                     |   característica (por ejemplo: dos telas    |
|                                     |   del mismo color).                         |
|                                     |                                             |
|                                     | **5. Habilidades de exploración y cuidado   |
|                                     | del entorno**                               |
|                                     |                                             |
|                                     | - Al utilizar sus sentidos para conocer     |
|                                     |   objetos, menciona palabras según su       |
|                                     |   sensación (por ejemplo: frío, caliente,   |
|                                     |   suave, rico).                             |
|                                     |                                             |
|                                     | - Muestra interés en conocer a los seres    |
|                                     |   vivos a través de sus sentidos.           |
|                                     |                                             |
|                                     | - Participa en cuidar a los seres vivos de  |
|                                     |   su ambiente.                              |
