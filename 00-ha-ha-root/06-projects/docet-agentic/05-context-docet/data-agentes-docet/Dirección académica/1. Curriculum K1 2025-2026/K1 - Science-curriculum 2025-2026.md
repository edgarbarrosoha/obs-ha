**II. LEARNING OUTCOMES AND SCOPE AND SEQUENCE**

+-------------------------------------+-----------------------------------------------------------------+
| **AREA**                            | **Learning Outcomes**                                           |
+-------------------------------------+-----------------------------------------------------------------+
|                                                                                                       |
+-------------------------------------+-----------------------------------------------------------------+
| Ciencias                            | ### 1. Habilidades de curiosidad y formulación de preguntas     |
|                                     |                                                                 |
|                                     | - Formula preguntas que expresan su curiosidad y su interés por |
|                                     |   saber más acerca de los seres vivos y el medio natural.       |
|                                     |                                                                 |
|                                     | ### 2. Habilidades de observación y descripción                 |
|                                     |                                                                 |
|                                     | - Reconoce y describe las características de los seres vivos.   |
|                                     |                                                                 |
|                                     | - Nota cambios en objetos, seres vivos y los comunica (por      |
|                                     |   ejemplo, cuando se da cuenta que han salido nuevas plantas en |
|                                     |   el huerto).                                                   |
|                                     |                                                                 |
|                                     | ### 3. Habilidades de experimentación y razonamiento científico |
|                                     |                                                                 |
|                                     | - Experimenta con diferentes elementos para dar respuesta a     |
|                                     |   problemas y preguntas acerca del mundo natural (por ejemplo:  |
|                                     |   poniendo objetos en el agua para ver cuáles flotan y cuáles   |
|                                     |   no).                                                          |
|                                     |                                                                 |
|                                     | - Formula hipótesis en los experimentos.                        |
|                                     |                                                                 |
|                                     | - Demuestra atención y hace comentarios apropiados durante los  |
|                                     |   experimentos de ciencias.                                     |
|                                     |                                                                 |
|                                     | ### 4. Habilidades de respeto y cuidado ambiental               |
|                                     |                                                                 |
|                                     | - Reconoce la importancia del sol, el agua y el aire para la    |
|                                     |   vida, y manifiesta actitudes de respeto y cuidado.            |
|                                     |                                                                 |
|                                     | - Participa cuidando de los seres vivos (por ejemplo: dándoles  |
|                                     |   de comer a los animales de la granja, regando plantas).       |
|                                     |                                                                 |
|                                     | ### 5. Habilidades de comunicación y socialización científica   |
|                                     |                                                                 |
|                                     | - Compara y comparte la información de sus hallazgos durante    |
|                                     |   los experimentos.                                             |
