**II. LEARNING OUTCOMES AND SCOPE AND SEQUENCE**

+-------------------------------------+------------------------------------------------------+
| **AREA**                            | **Learning Outcomes**                                |
+-------------------------------------+------------------------------------------------------+
|                                                                                            |
+-------------------------------------+------------------------------------------------------+
| Socioemocional                      | ### 1. Habilidades de convivencia y respeto          |
|                                     |                                                      |
|                                     | - Convive de manera respetuosa y armoniosa con sus   |
|                                     |   maestros.                                          |
|                                     |                                                      |
|                                     | - Participa al establecer los acuerdos del salón,    |
|                                     |   propiciando una convivencia sana.                  |
|                                     |                                                      |
|                                     | - Respeta los acuerdos del salón.                    |
|                                     |                                                      |
|                                     | - Juega armoniosamente en pequeños grupos.           |
|                                     |                                                      |
|                                     | - Se pone de acuerdo con sus compañeros para         |
|                                     |   realizar una actividad (por ejemplo, \"Vamos a     |
|                                     |   jugar al gato y al ratón; yo soy el gato y ustedes |
|                                     |   los ratones\").                                    |
|                                     |                                                      |
|                                     | - Dice \"Por favor\" y \"Gracias\" espontáneamente.  |
|                                     |                                                      |
|                                     | - Hace cosas para el grupo (por ejemplo, repartir    |
|                                     |   los platos y los vasos).                           |
|                                     |                                                      |
|                                     | - Llega puntualmente a clases todos los días.        |
|                                     |                                                      |
|                                     | - Porta el uniforme limpio y completo.               |
|                                     |                                                      |
|                                     | - Aspecto personal como uñas limpias y cortadas.     |
|                                     |                                                      |
|                                     | ### 2. Habilidades de autorregulación y atención     |
|                                     |                                                      |
|                                     | - Sabe escuchar en el tiempo de grupo a sus          |
|                                     |   compañeros, esperando su turno para participar.    |
|                                     |                                                      |
|                                     | - Termina las actividades que inicia.                |
|                                     |                                                      |
|                                     | - Puede mantener su atención cuando está trabajando. |
|                                     |                                                      |
|                                     | - Reconoce lo que puede hacer con y sin ayuda y      |
|                                     |   solicita ayuda cuando la necesita.                 |
|                                     |                                                      |
|                                     | - Es responsable en la realización de sus tareas.    |
|                                     |                                                      |
|                                     | ### 3. Habilidades de expresión emocional y empatía  |
|                                     |                                                      |
|                                     | - Expresa sus sentimientos.                          |
|                                     |                                                      |
|                                     | - Reconoce los sentimientos básicos de los demás     |
|                                     |   (enojo, tristeza, amor).                           |
|                                     |                                                      |
|                                     | - Intercambia expresiones de afecto con la maestra.  |
|                                     |                                                      |
|                                     | - Intercambia expresiones de afecto con los          |
|                                     |   compañeros.                                        |
|                                     |                                                      |
|                                     | ### 4. Habilidades de autonomía y toma de decisiones |
|                                     |                                                      |
|                                     | - Realiza por sí mismo acciones de cuidado personal, |
|                                     |   se hace cargo de sus pertenencias y respeta las de |
|                                     |   los demás.                                         |
|                                     |                                                      |
|                                     | - Elige los recursos que necesita para realizar      |
|                                     |   diferentes actividades.                            |
