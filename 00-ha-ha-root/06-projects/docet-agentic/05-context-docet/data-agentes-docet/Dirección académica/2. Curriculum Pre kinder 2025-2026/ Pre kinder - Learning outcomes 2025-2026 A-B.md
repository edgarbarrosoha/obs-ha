+---------------------------------------------------------------------------+
| El curriculum de preescolar es un continuo, lo que significa que los      |
| aprendizajes se construyen de manera progresiva, desde el nivel de        |
| Nursery hasta Kinder 3. A lo largo de esta etapa, cada objetivo se va     |
| fortaleciendo y profundizando para que los niños adquieran las            |
| habilidades necesarias para su siguiente etapa educativa.                 |
|                                                                           |
| Durante cada ciclo escolar, los objetivos académicos y de desarrollo      |
| están definidos de manera clara. Estos objetivos no cambian a lo largo    |
| del año, sino que se trabajan de forma continua. La forma en que se       |
| abordan se ajusta al nivel de dificultad y complejidad para desafiar a    |
| los alumnos a medida que avanzan. Este enfoque garantiza que, al          |
| finalizar el año, se logre la consolidación de los aprendizajes           |
| esperados, proporcionando una base sólida para su desarrollo integral.    |
