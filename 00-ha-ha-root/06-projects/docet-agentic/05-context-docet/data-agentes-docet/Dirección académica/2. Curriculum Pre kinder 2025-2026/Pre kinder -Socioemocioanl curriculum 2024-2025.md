|                                     | - Saluda al llegar.                         |

|                                     | - Participa en la acogida.                  |
|                                     |                                             |
|                                     | - Respeta los acuerdos del salón.           |
|                                     |                                             |
|                                     | - Comienza a esperar su turno.              |
|                                     |                                             |
|                                     | - Utiliza las frases \"por favor\" y        |

|                                     | - Invita a algún compañero o amigo a jugar  |

|                                     | - Sabe escuchar en el tiempo de grupo a sus |
|                                     |   compañeros, esperando su turno para       |
|                                     |   participar.                               |
|                                     |                                             |
|                                     | - Hace cosas para el grupo (por ejemplo,    |
|                                     |   repartir los platos y los vasos).         |
|                                     |                                             |
|                                     | **2. Habilidades emocionales y de           |
|                                     | autorregulación**                           |
|                                     |                                             |
|                                     | - Permite ser consolado por sus maestros    |
|                                     |   (por ejemplo, cuando se lastima permite   |
|                                     |   que lo abracen y muestra lo que le        |
|                                     |   sucedió).                                 |
|                                     |                                             |
|                                     | - Comienza a identificar sus propios        |
|                                     |   sentimientos.                             |
|                                     |                                             |
|                                     | - Se interesa en las necesidades de los     |
|                                     |   otros.                                    |
|                                     |                                             |
|                                     | - Muestra empatía por los sentimientos de   |
|                                     |   los demás.                                |
|                                     |                                             |
|                                     | - Puede mantener su atención cuando está    |
|                                     |   trabajando.                               |
|                                     |                                             |
|                                     | **3. Habilidades de autonomía y toma de     |
|                                     | decisiones**                                |
|                                     |                                             |
|                                     | - Participa activamente en las actividades. |
|                                     |                                             |
|                                     | - Elige los recursos que necesita para      |
|                                     |   realizar diferentes actividades.          |
