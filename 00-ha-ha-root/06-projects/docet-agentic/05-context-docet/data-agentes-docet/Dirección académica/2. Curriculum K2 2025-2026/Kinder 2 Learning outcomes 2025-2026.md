+-------------------------------------+-------------------------------------------------------------+

+-------------------------------------+-------------------------------------------------------------+

+-------------------------------------+-------------------------------------------------------------+
| Desarrollo Socioemocional           | **Autonomia e independencia**                               |
|                                     |                                                             |
|                                     | - Termina a tiempo las actividades que inicia               |
|                                     |                                                             |
|                                     | - Manifiesta motivación intrínseca                          |
|                                     |                                                             |
|                                     | - Se ajusta a nuevas situaciones (por ejemplo: si hay un    |
|                                     |   cambio en la rutina o el horario).                        |
|                                     |                                                             |
|                                     | - Muestra confianza e independencia.                        |
|                                     |                                                             |
|                                     | - Toma decisiones independientes para elegir actividades    |
|                                     |   durante el trabajo personal.                              |
|                                     |                                                             |
|                                     | - Mantiene la concentración e interés para acabar una       |
|                                     |   actividad a pesar de que haya distracciones.              |
|                                     |                                                             |
|                                     | - Demuestra hábitos de higiene personal y presentación      |
|                                     |   adecuada (uniforme completo, peinado).                    |
|                                     |                                                             |
|                                     | - Es responsable con el cumplimiento de sus tareas.         |
|                                     |                                                             |
|                                     | **Autoconocimiento y autoestima**                           |
|                                     |                                                             |
|                                     | - Muestra seguridad en sí mismo                             |
|                                     |                                                             |
|                                     | - Expresa verbalmente sus sentimientos                      |
|                                     |                                                             |
|                                     | - Defiende a sus compañeros cuando es necesario.            |
|                                     |                                                             |
|                                     | **Autorregulación emocional**                               |
|                                     |                                                             |
|                                     | - Regula sus emociones con herramientas como: patear la     |
|                                     |   pelota, respirar, botella mágica, etc.                    |
|                                     |                                                             |
|                                     | - Reconoce los sentimientos de los demás (por ejemplo:      |
|                                     |   \"Pablo está triste porque se le perdió su juguete\").    |
|                                     |                                                             |
|                                     | - Repara el daño que ocasionó (por ejemplo: limpia la mesa  |
|                                     |   si derramó limonada).                                     |
|                                     |                                                             |
|                                     | **Comunicación y habilidades sociales**                     |
|                                     |                                                             |
|                                     | - Se integra al juego grupal                                |
|                                     |                                                             |
|                                     | - Se pone de acuerdo con sus compañeros para realizar una   |
|                                     |   actividad (por ejemplo: \"vamos a jugar al gato y al      |
|                                     |   ratón; yo soy el gato y ustedes los ratones\").           |
|                                     |                                                             |
|                                     | - En ocasiones puede jugar con sus compañeros cediendo en   |
|                                     |   sus preferencias personales (por ejemplo: \"acepto que    |
|                                     |   hoy yo no soy el doctor\").                               |
|                                     |                                                             |
|                                     | - Sabe escuchar en el tiempo de grupo a sus compañeros,     |
|                                     |   esperando su turno para participar                        |
|                                     |                                                             |
|                                     | - Intercambia expresiones de afecto con la maestra.         |
|                                     |                                                             |
|                                     | - Intercambia expresiones de afecto con los compañeros.     |
|                                     |                                                             |
|                                     | - Puede defenderse verbalmente estableciendo límites cuando |
|                                     |   es necesario (por ejemplo al decir: \"no me gusta\").     |
|                                     |                                                             |
|                                     | **Convivencia y trabajo colaborativo**                      |
|                                     |                                                             |
|                                     | - Es capaz de solucionar problemas con sus compañeros       |
|                                     |   cuando hay alguna discusión o conflicto.                  |
|                                     |                                                             |
|                                     | - Puede resolver problemas sencillos buscando opciones y    |
|                                     |   experimentando posibles soluciones.                       |
|                                     |                                                             |
|                                     | - Trabaja junto con sus compañeros para lograr una tarea.   |
|                                     |                                                             |
|                                     | - Participa en la discusión para solucionar problemas del   |
|                                     |   grupo.                                                    |
|                                     |                                                             |
|                                     | - Colabora en las tareas del salón (por ejemplo: repartir   |
|                                     |   los platos y los vasos, limpiar la mesa, etc.).           |
|                                     |                                                             |
|                                     | - Cuida los materiales del salón de clases.                 |
|                                     |                                                             |
|                                     | **Sentido de responsabilidad y normas**                     |
|                                     |                                                             |
|                                     | - Tiene un concepto claro de autoridad                      |
|                                     |                                                             |
|                                     | - Respeta la figura del maestro                             |
|                                     |                                                             |
|                                     | - Respeta los acuerdos del salón                            |
|                                     |                                                             |
|                                     | - Participa al establecer los acuerdos del salón            |
|                                     |                                                             |
|                                     | - Sigue las rutinas del salón                               |
|                                     |                                                             |
|                                     | - Sigue instrucciones de un adulto                          |
|                                     |                                                             |
|                                     | - Comprende el proceso de votación, aceptando la decisión   |
|                                     |   mayoritaria.                                              |

|                                     |                                                             |
|                                     | - Corre alternando brazos y piernas.                        |

|                                     |                                                             |

|                                     |   pies.                                                     |
|                                     |                                                             |
|                                     | - Se siente motivado a saltar barreras.                     |
|                                     |                                                             |
|                                     | - Arroja las pelotas a lo alto.                             |
|                                     |                                                             |
|                                     | - Lanza una pelota a otra persona.                          |
|                                     |                                                             |
|                                     | - Emplea el lanzamiento unilateral (con una sola mano).     |
|                                     |                                                             |
|                                     | - Atrapa pelotas pequeñas, utilizando más las manos que los |
