| Pensamiento matemático              | FORMAS Y ESPACIO                            |
|                                     |                                             |
|                                     | - Utiliza términos relacionados al tiempo:  |
|                                     |   ayer, hoy y mañana.                       |
|                                     |                                             |
|                                     | - Logra hacer peticiones y seguir           |
|                                     |   indicaciones para colocar objetos según   |
|                                     |   las diferentes posiciones en el espacio:  |
|                                     |   arriba/abajo, enfrente/atrás,             |
|                                     |   debajo/sobre, cerca/lejos.                |
|                                     |                                             |
|                                     | - Reconoce las figuras geométricas:         |
|                                     |   círculo, cuadrado, triángulo, rectángulo, |
|                                     |   estrella, óvalo, rombo y corazón.         |
|                                     |                                             |
|                                     | <!-- -->                                    |
|                                     |                                             |
|                                     | - Clasifica y acomoda objetos dependiendo   |
|                                     |   de sus características (por ejemplo:      |
|                                     |   color, forma, tamaño, textura) y puede    |
|                                     |   basarse en dos condiciones (por ejemplo:  |
|                                     |   que sean círculos rojos).                 |
|                                     |                                             |
|                                     | NÚMERO                                      |
|                                     |                                             |
|                                     | - Nombra los números del 1 al 30.           |
|                                     |                                             |
|                                     | - Identifica los números del 1 al 20 por su |
|                                     |   nombre y grafía.                          |
|                                     |                                             |
|                                     | - Compara cantidades numéricas (por         |
|                                     |   ejemplo, 8 es menos que el 10).           |
|                                     |                                             |
|                                     | - Logra relacionar cantidad y símbolo del 1 |
|                                     |   al 20.                                    |
|                                     |                                             |
|                                     | - Identifica algunas relaciones de          |
|                                     |   equivalencia entre monedas de \$1, \$2 y  |
|                                     |   \$5 en situaciones reales o ficticias de  |
|                                     |   compra y venta.                           |
|                                     |                                             |
|                                     | PATRONES                                    |
|                                     |                                             |
|                                     | - Sigue la secuencia o patrón que se le da  |
|                                     |   y puede crear patrones por sí solo (por   |
|                                     |   ejemplo: legos amarillo, rojo, amarillo,  |
|                                     |   rojo).                                    |
|                                     |                                             |
|                                     | - Acomoda los objetos en una serie y        |
|                                     |   explica su relación.                      |
|                                     |                                             |
|                                     | MEDIDAS                                     |
|                                     |                                             |
|                                     | - [Mide objetos mediante el uso de medidas  |
|                                     |   no convencionales y puede comparar        |
|                                     |   objetos según su medida. (Por ejemplo:    |
|                                     |   Reconoce que una mesa es más larga que    |
|                                     |   otra).]{.mark}                            |
|                                     |                                             |
|                                     | RECOLECCIÓN Y ANÁLISIS DE DATOS             |
|                                     |                                             |
|                                     | - Comienza a utilizar una gráfica para      |
|                                     |   analizar datos y comunicar resultados.    |
|                                     |                                             |
|                                     | - Muestra una buena capacidad de memoria.   |
|                                     |                                             |
|                                     | - [Puede comparar objetos según su medida   |
|                                     |   (por ejemplo: reconoce que una mesa es    |
|                                     |   más larga que otra). (ES IGUAL)]{.mark}   |
|                                     |                                             |
|                                     | MATEMÁTICAS PARA LA CREATIVIDAD Y           |
|                                     | RESOLUCIÓN DE PROBLEMAS                     |
|                                     |                                             |
|                                     | - Sigue ruta en juegos de mesa (por         |
|                                     |   ejemplo: sigue el orden del juego de      |
|                                     |   serpientes y escaleras).                  |
|                                     |                                             |
|                                     | - Nombra el objeto que no corresponde en un |
|                                     |   grupo.                                    |
|                                     |                                             |
|                                     | - Da sentido a las construcciones que hace  |
|                                     |   con sus juguetes.                         |
|                                     |                                             |
|                                     | - Comprende lo que es una suma y puede      |
|                                     |   resolver sumas sencillas utilizando       |
|                                     |   objetos manipulables.                     |
|                                     |                                             |
|                                     | - Comprende lo que es una resta y puede     |
|                                     |   resolver restas sencillas utilizando      |
|                                     |   objetos manipulables.                     |
|                                     |                                             |
|                                     | - Arma rompecabezas de 20 a 24 piezas.      |
