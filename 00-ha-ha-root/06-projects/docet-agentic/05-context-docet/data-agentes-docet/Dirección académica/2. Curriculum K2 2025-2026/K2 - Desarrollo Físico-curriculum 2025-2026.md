+-------------------------------------+---------------------------------------------+
| **AREA**                            | **Learning Outcomes**                       |
+-------------------------------------+---------------------------------------------+
|                                                                                   |
+-------------------------------------+---------------------------------------------+
| Desarrollo Físico                   | MOTOR FINO                                  |
|                                     |                                             |
|                                     | - Respeta el contorno de los dibujos al     |
|                                     |   colorear.                                 |
|                                     |                                             |
|                                     | - Calca imágenes sencillas.                 |
|                                     |                                             |
|                                     | - Traza respetando espacios reducidos.      |
|                                     |                                             |
|                                     | - Puede delinear dibujos.                   |
|                                     |                                             |
|                                     | - Puede recortar siguiendo el contorno de   |
|                                     |   figuras definidas.                        |
|                                     |                                             |
|                                     | - Rasga papel utilizando los dedos          |
|                                     |   adecuados.                                |
|                                     |                                             |
|                                     | - Pasa granos con puño de un recipiente a   |
|                                     |   otro.                                     |
|                                     |                                             |
|                                     | - Pasa granos con 3 dedos de un recipiente  |
|                                     |   a otro.                                   |
|                                     |                                             |
|                                     | - Pasa granos con cuchara de un recipiente  |
|                                     |   a otro.                                   |
|                                     |                                             |
|                                     | - Pasa granos con jarras.                   |
|                                     |                                             |
|                                     | - Pasa agua con jarras.                     |
|                                     |                                             |
|                                     | - Pasa agua con gotero.                     |
|                                     |                                             |
|                                     | - Se limpia la nariz sin ayuda.             |
|                                     |                                             |
|                                     | - Se lava y seca las manos sin ayuda.       |
|                                     |                                             |
|                                     | MOTOR GRUESO                                |
|                                     |                                             |
|                                     | - Identifica sus posibilidades expresivas y |
|                                     |   motrices en actividades que implican      |
|                                     |                                             |
|                                     | - organización espacio-temporal,            |
|                                     |   lateralidad, equilibrio y coordinación.   |
|                                     |                                             |
|                                     | - Reconoce las características que lo       |
|                                     |   identifican y diferencian de los demás en |
|                                     |                                             |
|                                     | - actividades y juegos.                     |
|                                     |                                             |
|                                     | - Conoce y diferencia aspectos y partes     |
|                                     |   fundamentales de su propio cuerpo.        |
|                                     |                                             |
|                                     | - Mantiene el equilibrio en un pie.         |
|                                     |                                             |
|                                     | - Se desplaza de un punto a otro en línea   |
|                                     |   recta.                                    |
|                                     |                                             |
|                                     | - Conoce los conceptos arriba y abajo; a un |
|                                     |   lado y a otro; izquierda y derecha.       |
|                                     |                                             |
|                                     | - Brinca sobre un pie.                      |
|                                     |                                             |
|                                     | - Salta hacia atrás con los dos pies.       |
|                                     |                                             |
|                                     | - Camina en zig-zag esquivando conos.       |
|                                     |                                             |
|                                     | - Corre galopando.                          |
|                                     |                                             |
|                                     | - Patea una pelota en movimiento.           |
|                                     |                                             |
|                                     | - Corre alternando brazos y piernas.        |
|                                     |                                             |
|                                     | - Mientras va corriendo, cambia de          |
|                                     |   dirección sin perder el equilibrio.       |
|                                     |                                             |
|                                     | - Salta utilizando cada pie de modo         |
|                                     |   alternado.                                |
|                                     |                                             |
|                                     | - Salta desde una altura de 30 cm. y cae    |
|                                     |   sobre sus dos pies.                       |
|                                     |                                             |
|                                     | - Se siente motivado a saltar barreras.     |
|                                     |                                             |
|                                     | - Arroja las pelotas a lo alto.             |
|                                     |                                             |
|                                     | - Lanza una pelota a otra persona.          |
|                                     |                                             |
|                                     | - Emplea el lanzamiento unilateral (con una |
|                                     |   sola mano).                               |
|                                     |                                             |
|                                     | - Atrapa pelotas pequeñas, utilizando más   |
|                                     |   las manos que los brazos.                 |
|                                     |                                             |
|                                     | - Bota una pelota con ambas manos.          |
|                                     |                                             |
|                                     | - Rueda hacia un punto determinado.         |
|                                     |                                             |
|                                     | Intenta hacer rueda de coche.               |
|                                     |                                             |
|                                     | Sigue las reglas durante los juegos de      |
|                                     | actividad física.                           |
|                                     |                                             |
|                                     | Utiliza expresiones corporales.             |
