| **AREA**                            | **Learning Outcomes**                       |
+-------------------------------------+---------------------------------------------+
|                                                                                   |
+-------------------------------------+---------------------------------------------+
| Lenguaje                            | Receptivo                                   |
|                                     |                                             |
|                                     | - Escucha atentamente.                      |
|                                     |                                             |
|                                     | - Sigue instrucciones que impliquen tres o  |
|                                     |   cuatro acciones.                          |
|                                     |                                             |
|                                     | - Relaciona información previa con          |
|                                     |   información nueva haciendo conexiones.    |
|                                     |                                             |
|                                     | - Sus aportaciones o comentarios son        |
|                                     |   relacionados con el tema que se está      |
|                                     |   tratando.                                 |
|                                     |                                             |
|                                     | - Sigue con fluidez la conversación de un   |
|                                     |   adulto o compañero.                       |
|                                     |                                             |
|                                     | - Pregunta por el significado de palabras.  |
|                                     |                                             |
|                                     | - Comprende el concepto de opuestos y puede |
|                                     |   decir el opuesto de alguna palabra.       |
|                                     |                                             |
|                                     | Expresivo                                   |
|                                     |                                             |
|                                     | - Articula adecuadamente la mayoría de los  |
|                                     |   fonemas.                                  |
|                                     |                                             |
|                                     | - Habla con claridad y estructura           |
|                                     |   adecuadamente las oraciones.              |
|                                     |                                             |
|                                     | - Conversa en el tiempo de grupo, aportando |
|                                     |   sus ideas o respondiendo preguntas.       |
|                                     |                                             |
|                                     | - Expresa experiencias anteriores en el     |
|                                     |   orden que sucedieron (por ejemplo,        |
|                                     |   \"primero fui con mi mamá al súper y      |
|                                     |   después fuimos a ver a mi abuelita\").    |
|                                     |                                             |
|                                     | - Hace peticiones específicas (por ejemplo: |
|                                     |   \"¿me puedes prestar el color azul, por   |
|                                     |   favor?).                                  |
|                                     |                                             |
|                                     | - Expresa las similitudes y diferencias     |
|                                     |   entre objetos.                            |
|                                     |                                             |
|                                     | - Usa objetos simbólicos como objetos       |
|                                     |   reales y se expresa sobre ello.           |
+-------------------------------------+---------------------------------------------+
| Lectoescritura                      | ESCRITURA                                   |
|                                     |                                             |
|                                     | - Comienza a definir su lateralidad.        |
|                                     |                                             |
|                                     | - Toma el lápiz adecuadamente.              |
|                                     |                                             |
|                                     | - Respeta el contorno de los dibujos para   |
|                                     |   colorear y delinear.                      |
|                                     |                                             |
|                                     | - Puede diferenciar entre letras y números. |
|                                     |                                             |
|                                     | - Pregunta cómo escribir letras y números.  |
|                                     |                                             |
|                                     | - Traza adecuadamente letras y números.     |
|                                     |                                             |
|                                     | - Copia números.                            |
|                                     |                                             |
|                                     | - Copia palabras relacionadas a los         |
|                                     |   proyectos.                                |
|                                     |                                             |
|                                     | - Escribe letras reconocibles por sí solo.  |
|                                     |                                             |
|                                     | - Logra escribir su nombre por sí solo.     |
|                                     |                                             |
|                                     | - Entiende que el propósito de la escritura |
|                                     |   es transmitir un mensaje.                 |
|                                     |                                             |
|                                     | - Se inicia en la escritura de las letras   |
|                                     |   del abecedario por su sonido.             |
|                                     |                                             |
|                                     | - [​​Produce dibujos para informar algo de    |
|                                     |   interés.]{.mark}                          |
|                                     |                                             |
|                                     | LECTURA                                     |
|                                     |                                             |
|                                     | - Entiende que los libros, revistas,        |
|                                     |   periódicos y otros formatos impresos      |
|                                     |   tienen un mensaje (por ejemplo: acude a   |
|                                     |   los libros para realizar una              |
|                                     |   investigación).                           |
|                                     |                                             |
|                                     | - Demuestra comprensión de las historias    |
|                                     |   que son leídas, haciendo preguntas y      |
|                                     |   comentarios sobre ellas.                  |
|                                     |                                             |
|                                     | - Tiene habilidad para volver a contar      |
|                                     |   historias con sus propias palabras.       |
|                                     |                                             |
|                                     | - Puede ordenar en secuencia lógica un      |
|                                     |   cuento de tres a cinco partes.            |
|                                     |                                             |
|                                     | - Tiene habilidad para predecir el final de |
|                                     |   una historia.                             |
|                                     |                                             |
|                                     | - Compara o hace conexiones entre           |
|                                     |   diferentes historias o entre eventos      |
|                                     |   dentro de una misma historia.             |
|                                     |                                             |
|                                     | - Lee su nombre. ( Identificar las letras   |
|                                     |   de su nombre y poder armarlo)             |
|                                     |                                             |
|                                     | - Reconoce el nombre de sus compañeros por  |
|                                     |   escrito.                                  |
|                                     |                                             |
|                                     | - Establece correspondencia entre el        |
|                                     |   lenguaje oral y escrito.                  |
|                                     |                                             |
|                                     | - [Reconoce el sonido de algunas letras del |
|                                     |   abecedario (al menos 10 letras).]{.mark}  |
|                                     |                                             |
|                                     | - Logra mencionar palabras que inicien con  |
|                                     |   el sonido que se indica.                  |
|                                     |                                             |
|                                     | - Reconoce las palabras del vocabulario de  |
|                                     |   los proyectos y las puede ubicar          |
|                                     |   adecuadamente en la \"pared de            |
|                                     |   palabras\".                               |
|                                     |                                             |
|                                     | - Logra relacionar las palabras             |
|                                     |   relacionadas a los proyectos con las      |
|                                     |   imágenes que las representan.             |
