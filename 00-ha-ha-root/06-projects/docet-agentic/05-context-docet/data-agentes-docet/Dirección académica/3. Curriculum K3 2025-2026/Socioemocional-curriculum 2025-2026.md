**II. LEARNING OUTCOMES AND SCOPE AND SEQUENCE**

+-------------------------------------+---------------------------------------------+
| **AREA**                            | **Learning Outcomes**                       |
+-------------------------------------+---------------------------------------------+
|                                                                                   |
+-------------------------------------+---------------------------------------------+
| Socioemocional                      | - Reconoce y expresa características        |
|                                     |   personales: nombre, cómo es físicamente,  |
|                                     |   qué le gusta, qué no le gusta, qué se le  |
|                                     |   dificulta y qué se le facilita.           |
|                                     |                                             |
|                                     | - Respeta las figuras de autoridad.         |
|                                     |                                             |
|                                     | - Respeta los acuerdos del salón.           |
|                                     |                                             |
|                                     | - Manifiesta motivación intrínseca.         |
|                                     |                                             |
|                                     | - Persiste en la realización de actividades |
|                                     |   que le impliquen un reto o esfuerzo       |
|                                     |   mayor.                                    |
|                                     |                                             |
|                                     | - Se expresa con seguridad y defiende sus   |
|                                     |   ideas ante sus compañeros.                |
|                                     |                                             |
|                                     | - Escucha a sus compañeros y espera su      |
|                                     |   turno.                                    |
|                                     |                                             |
|                                     | - Colabora en actividades del grupo y       |
|                                     |   escolares, propone ideas y considera las  |
|                                     |   de los demás cuando participa en          |
|                                     |   actividades en equipo y en grupo.         |
|                                     |                                             |
|                                     | - Se integra al juego grupal.               |
|                                     |                                             |
|                                     | - Propone acuerdos para la convivencia, el  |
|                                     |   juego o el trabajo, explica su utilidad y |
|                                     |   actúa con apego a ellos.                  |
|                                     |                                             |
|                                     | - Puede establecer acuerdos con sus         |
|                                     |   compañeros (por ejemplo: tu eres el       |
|                                     |   doctor y yo soy el enfermo, después tu    |
|                                     |   eres el enfermo y yo el doctor.)          |
|                                     |                                             |
|                                     | - Comienza a compartir sus cosas con sus    |
|                                     |   compañeros.                               |
|                                     |                                             |
|                                     | - Reconoce y nombra situaciones que le      |
|                                     |   generan alegría, seguridad, tristeza,     |
|                                     |   miedo o enojo y expresa lo que siente.    |
|                                     |                                             |
|                                     | - Canaliza adecuadamente sus emociones.     |
|                                     |                                             |
|                                     | - Reconoce los sentimientos de los demás.   |
|                                     |                                             |
|                                     | - Reconoce cuando alguien necesita ayuda y  |
|                                     |   la proporciona.                           |
|                                     |                                             |
|                                     | - Habla de sus conductas y las de otros y   |
|                                     |   explica las consecuencias de algunas de   |
|                                     |   ellas para relacionarse con otros.        |
|                                     |                                             |
|                                     | - Repara el daño que ocasionó ( Por         |
|                                     |   ejemplo: \"Limpia la mesa si derramó      |
|                                     |   limonada\").                              |
|                                     |                                             |
|                                     | - Realiza con gusto encargos en el salón    |
|                                     |   (por ejemplo: Repartir los platos y los   |
|                                     |   vasos).                                   |
|                                     |                                             |
|                                     | - Es responsable del cuidado del salón y    |
|                                     |   los materiales.                           |
|                                     |                                             |
|                                     | - Se adapta a nuevas situaciones (por       |
|                                     |   ejemplo: si hay un cambio en la rutina o  |
|                                     |   el horario).                              |
|                                     |                                             |
|                                     | - Demuestra confianza en las maestras y     |
|                                     |   sabe que son personas que lo pueden       |
|                                     |   ayudar.                                   |
|                                     |                                             |
|                                     | - Es capaz de solucionar problemas con sus  |
|                                     |   compañeros cuando hay alguna discusión o  |
|                                     |   conflicto.                                |
|                                     |                                             |
|                                     | - Comprende el proceso de votación          |
|                                     |   aceptando la decisión mayoritaria.        |
|                                     |                                             |
|                                     | - Manifiesta actitudes y relaciones de      |
|                                     |   respeto y colaboración.                   |
|                                     |                                             |
|                                     | - Elige el área y las actividades en las    |
|                                     |   que quiere trabajar durante trabajo       |
|                                     |   personal.                                 |
|                                     |                                             |
|                                     | - Demuestra hábitos de higiene personal y   |
|                                     |   presentación adecuada (Uniforme completo, |
|                                     |   peinado)                                  |
|                                     |                                             |
|                                     | - Es responsable con el cumplimiento de sus |
|                                     |   tareas                                    |
