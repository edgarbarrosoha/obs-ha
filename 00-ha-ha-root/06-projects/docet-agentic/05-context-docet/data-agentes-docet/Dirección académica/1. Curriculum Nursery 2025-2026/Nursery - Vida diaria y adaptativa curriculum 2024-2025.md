|                                     |                                             |
|                                     | - Vestirse y desvestirse parcialmente       |
|                                     |   (estirar brazos, quitarse zapatos,        |
|                                     |   intentar ponérselos).                     |
|                                     |                                             |
|                                     | - Higiene personal básica (sonarse la       |
|                                     |   nariz, limpiarse la cara, avisar          |
|                                     |   necesidad de cambio de pañal).            |
|                                     |                                             |
|                                     | - Lavado de manos con ayuda.                |
|                                     |                                             |
|                                     | - Control de esfínteres inicial (avisa      |
|                                     |   cuando el pañal está sucio o necesita     |
|                                     |   cambio).                                  |
|                                     |                                             |
|                                     | **2. Habilidades de alimentación            |
|                                     | independiente**                             |
|                                     |                                             |
|                                     | - Masticar alimentos sólidos.               |
|                                     |                                             |
|                                     | - Sostener el vaso con ambas manos.         |
|                                     |                                             |
|                                     | - Pedir comida o bebida cuando tiene hambre |
|                                     |   o sed.                                    |
|                                     |                                             |
|                                     | - Comer sin ayuda.                          |
|                                     |                                             |
|                                     | **3. Habilidades de colaboración y          |
|                                     | responsabilidad**                           |
|                                     |                                             |
|                                     | - Ayudar a poner la mesa y repartir         |
|                                     |   alimentos.                                |
|                                     |                                             |
|                                     | - Llevar platos y utensilios a la cocina.   |
|                                     |                                             |
|                                     | - Guardar sus recipientes en la lonchera    |
|                                     |   después del snack.                        |
|                                     |                                             |
|                                     | - Colocar pertenencias en su lugar.         |
|                                     |                                             |
|                                     | - Cooperar en las labores del salón         |
|                                     |   (recoger o repartir material).            |
|                                     |                                             |
|                                     | **4. Habilidades sociales y comunicativas   |
|                                     | adaptativas**                               |
|                                     |                                             |
|                                     | - Buscar ayuda cuando enfrenta un problema. |
|                                     |                                             |
|                                     | - Comunicar necesidades básicas (hambre,    |
|                                     |   sed, higiene)                             |
