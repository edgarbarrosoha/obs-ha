**II. LEARNING OUTCOMES AND SCOPE AND SEQUENCE**

+-------------------------------------+---------------------------------------------+
| **AREA**                            | **Learning Outcomes**                       |
+-------------------------------------+---------------------------------------------+
|                                                                                   |
+-------------------------------------+---------------------------------------------+
| Desarrollo Físico                   | MOTOR FINO                                  |
|                                     |                                             |
|                                     | - Mete y saca objetos de algún contenedor   |
|                                     |                                             |
|                                     | - Utiliza la cuchara o el tenedor para      |
|                                     |   llevarse comida a la boca.                |
|                                     |                                             |
|                                     | - Arma torres con legos y bloques de        |
|                                     |   madera.                                   |
|                                     |                                             |
|                                     | - Desarma el material de construcción (por  |
|                                     |   ejemplo: bloques de lego). Pasa de una    |
|                                     |   por una las hojas de cartón o de tela de  |
|                                     |   un libro.                                 |
|                                     |                                             |
|                                     | - Arma rompecabezas que impliquen ensartar  |
|                                     |   las piezas en su espacio                  |
|                                     |                                             |
|                                     | - correspondiente (por ejemplo: poner el    |
|                                     |   cuadrado en el espacio del cuadrado).     |
|                                     |                                             |
|                                     | - Exprime una esponja.                      |
|                                     |                                             |
|                                     | - Garabatea sobre papel con una crayola o   |
|                                     |   gis grueso.                               |
|                                     |                                             |
|                                     | - Pasa granos con puño.                     |
|                                     |                                             |
|                                     | - Ensarta objetos.                          |
|                                     |                                             |
|                                     | MOTOR GRUESO                                |
|                                     |                                             |
|                                     | - Repta coordinadamente: patrón cruzado.    |
|                                     |                                             |
|                                     | - Gatea coordinadamente: patrón cruzado.    |
|                                     |                                             |
|                                     | - Camina solo.                              |
|                                     |                                             |
|                                     | - Camina y se detiene sin perder el         |
|                                     |   equilibrio.                               |
|                                     |                                             |
|                                     | - Se pone en cunclillas y se incorpora sin  |
|                                     |   perder el equilibrio.                     |
|                                     |                                             |
|                                     | - Comienza a caminar de diferentes maneras  |
|                                     |   (marchando o hacia atrás).                |
|                                     |                                             |
|                                     | - Empuja y jala materiales al caminar       |
|                                     |                                             |
|                                     | - Se mueve al ritmo de la música.           |
|                                     |                                             |
|                                     | - Imita movimientos.                        |
|                                     |                                             |
|                                     | - Corre sin perder el equilibrio.           |
|                                     |                                             |
|                                     | - Brinca con los dos pies.                  |
|                                     |                                             |
|                                     | - Sube y baja escaleras tomado del          |
|                                     |   barandal. Permite mecerse (por ejemplo:   |
|                                     |   en una hamaca, columpio, paracaídas).     |
|                                     |   Patea la pelota.                          |
|                                     |                                             |
|                                     | - Lanza objetos utilizando indistintamente  |
|                                     |   las manos.                                |
|                                     |                                             |
|                                     | - Se puede sentar en la sillita sin ayuda.  |
|                                     |                                             |
|                                     | - Intenta empujarse en un carrito montable. |
