**II. APRENDIZAJES ESPERADOS Y SECUENCIA**

+-------------------------------------+---------------------------------------------+
| **AREA**                            | **APRENDIZAJES ESPERADOS**                  |
+-------------------------------------+---------------------------------------------+
|                                                                                   |
+-------------------------------------+---------------------------------------------+
| Sensorial                           | **1. Habilidades visuales**                 |
|                                     |                                             |
|                                     | - Le gusta mirar su reflejo en el espejo.   |
|                                     |                                             |
|                                     | - Discrimina visualmente objetos conocidos. |
|                                     |                                             |
|                                     | - Observa con atención objetos en           |
|                                     |   movimiento (por ejemplo, el movimiento de |
|                                     |   un carrito o una pelota).                 |
|                                     |                                             |
|                                     | **2. Habilidades táctiles**                 |
|                                     |                                             |
|                                     | - Disfruta tocar diferentes texturas.       |
|                                     |                                             |
|                                     | - Distingue formas u objetos a través del   |
|                                     |   tacto e intenta nombrarlas.               |
|                                     |                                             |
|                                     | - Distingue a través del tacto texturas     |
|                                     |   ásperas y lisas.                          |
|                                     |                                             |
|                                     | - Distingue temperaturas a través del       |
|                                     |   tacto: frío y caliente.                   |
|                                     |                                             |
|                                     | **3. Habilidades olfativas**                |
|                                     |                                             |
|                                     | - Utiliza su sentido del olfato para        |
|                                     |   conocer diferentes olores del ambiente.   |
|                                     |                                             |
|                                     | **4. Habilidades multisensoriales y de      |
|                                     | exploración del entorno**                   |
|                                     |                                             |
|                                     | - Muestra interés en conocer a los seres    |
|                                     |   vivos a través de sus sentidos.           |
|                                     |                                             |
|                                     | - Participa en cuidar a los seres vivos de  |
|                                     |   su ambiente.                              |
|                                     |                                             |
|                                     | **5. Habilidades auditivas**                |
|                                     |                                             |
|                                     | - Distingue los sonidos del ambiente y      |
|                                     |   nombra el objeto o ser vivo que los       |
|                                     |   genera.                                   |
|                                     |                                             |
|                                     | **6. Habilidades de asociación sensorial**  |
|                                     |                                             |
|                                     | - Puede relacionar dos objetos con la misma |
|                                     |   característica (por ejemplo: dos telas    |
|                                     |   del mismo color)                          |
