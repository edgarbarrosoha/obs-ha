|                                     |                                                      |
|                                     | - Realiza comentarios de los textos ilustrados.      |
|                                     |                                                      |

|                                     |                                                      |
|                                     | - Comprende el contenido de textos sencillos.        |
|                                     |                                                      |
|                                     | - Lee sílabas con vocales y consonantes variantes y  |
|                                     |   complejas.                                         |
|                                     |                                                      |
|                                     | - Sigue el programa de lectura que mejora la         |
|                                     |   comprensión, la fluidez y el pensamiento crítico.  |
|                                     |                                                      |

|                                     | - Copia textos sencillos.                            |
|                                     |                                                      |
|                                     | - Escribe palabras de manera espontánea.             |
|                                     |                                                      |
|                                     | - Redacta mensajes utilizando dibujos y palabras.    |
|                                     |                                                      |
|                                     | -                                                    |
|                                     |                                                      |
|                                     | Reflexión sobre la lengua                            |

|                                     |   oración]{.mark} .                                  |
|                                     |                                                      |

|                                     |   oración (Artículo, sustantivo, Adjetivo y verbo    |

| English                             | **Reading :**                                        |
|                                     |                                                      |
|                                     | - Reads predictable and simple patterned texts       |
|                                     |   independently.                                     |
|                                     |                                                      |
|                                     | - Makes predictions and identifies the main idea and |
|                                     |   key details.                                       |
|                                     |                                                      |
|                                     | - Compares characters and events across different    |
|                                     |   stories.                                           |
|                                     |                                                      |
|                                     | - Follows the reading program that enhances          |
|                                     |   comprehension, fluency and critical thinking.      |
|                                     |                                                      |
|                                     | **Phonics, phonological awareness and phonetic       |
|                                     | awareness:**                                         |
|                                     |                                                      |
|                                     | - Recognizes and decodes words with consonant        |
|                                     |   digraphs                                           |
|                                     |                                                      |
|                                     | - Uses phoneme blending and segmentation to write    |
|                                     |   new words.                                         |
|                                     |                                                      |
|                                     | - Begins to read and spell high-frequency sight      |
|                                     |   words.                                             |
|                                     |                                                      |
|                                     | **Writing:**                                         |
|                                     |                                                      |
|                                     | - Writes 2--3 connected sentences on a single topic. |
|                                     |                                                      |
|                                     | - Writes adjectives and verbs in a sentence.         |
|                                     |                                                      |
|                                     | - Uses question marks and exclamation marks          |
|                                     |   appropriately in short texts.                      |
|                                     |                                                      |
|                                     | **Grammar:**                                         |
|                                     |                                                      |
|                                     | - Identifies nouns and proper nouns.                 |
|                                     |                                                      |
|                                     | - Identifies adjectives to describe size, color, and |
|                                     |   emotion.                                           |
|                                     |                                                      |
|                                     | - Identifies and writes action verbs in present      |
|                                     |   tense.                                             |
|                                     |                                                      |
|                                     | - Uses the verb to be in written sentences to        |
|                                     |   describe people, places, and feelings.             |
|                                     |                                                      |
|                                     | - Reads and identifies sight words within short      |
|                                     |   texts.                                             |
|                                     |                                                      |
|                                     | **Speaking/Listening**                               |
|                                     |                                                      |
|                                     | - Listens actively and responds to simple questions. |
|                                     |                                                      |
|                                     | - Expresses personal opinions using sentence         |
|                                     |   starters.                                          |
|                                     |                                                      |
|                                     | - Asks and answers interrogative questions related   |
|                                     |   to classroom topics.                               |
+-------------------------------------+------------------------------------------------------+
| Math                                | **Number Concepts and Operations**                   |
|                                     |                                                      |
|                                     | - **Building Numbers to 0- 100**                     |
|                                     |                                                      |
|                                     | - Practices skip counting.                           |
|                                     |                                                      |
|                                     | - Understands the place value.                       |
|                                     |                                                      |
|                                     | **Addition and Subtraction (0--100)**                |
|                                     |                                                      |
|                                     | - Practices mental and written addition and          |
|                                     |   subtraction.                                       |
|                                     |                                                      |
|                                     | - Uses number lines.                                 |
|                                     |                                                      |
|                                     | - Creates and solves real-life problems.             |
|                                     |                                                      |
|                                     | **Comparing and Ordering**                           |
|                                     |                                                      |
|                                     | - Compares numbers using more than/less than.        |
|                                     |                                                      |
|                                     | - Orders numbers up to 100 in ascending/descending   |
|                                     |   order.                                             |
|                                     |                                                      |
|                                     | - Identifies odd and even numbers.                   |
|                                     |                                                      |
|                                     | **Number Patterns and Sequences**                    |
|                                     |                                                      |
|                                     | - Counts by 1s, 2s, 3s, 5s, and 10s (including       |
|                                     |   odd/even) up to 100.                               |
|                                     |                                                      |
|                                     | - Identifies, completes, and creates number          |
|                                     |   patterns.                                          |
|                                     |                                                      |
|                                     | - Represents number sequences graphically and        |
|                                     |   performs sequential operations up to 100.          |
|                                     |                                                      |
|                                     | **Measurement and Time**                             |
|                                     |                                                      |
|                                     | - Reads anologl clocks; tells time in minutes,       |
|                                     |   hours, days, and months.                           |
|                                     |                                                      |
|                                     | - Understands elapsed time and Gantt diagrams;       |
|                                     |   identifies days, months, and seasons.              |
|                                     |                                                      |
|                                     | **Length and Weight**:                               |
|                                     |                                                      |
|                                     | - Measures and compares lengths, heights, and mass;  |
|                                     |   records measurements using appropriate units and   |
|                                     |   tools.                                             |
|                                     |                                                      |
|                                     | **Geometry**                                         |
|                                     |                                                      |
|                                     | - Identifies and classifies 2D and 3D shapes         |
|                                     |   (circle, square, prism, cube, etc.).               |
|                                     |                                                      |
|                                     | - Understands concepts of regular/irregular and      |
|                                     |   symmetry/asymmetry.                                |
|                                     |                                                      |
|                                     | - Counts edges, faces, and vertices on 3D shapes.    |
|                                     |                                                      |
|                                     | - Recognizes shape movements (slides, flips, turns)  |
|                                     |   and models 3D objects digitally.                   |
|                                     |                                                      |
|                                     | - Recognizes and classifies open and closed 2D       |
|                                     |   figures (circle, square, triangle, rectangle,      |
|                                     |   etc.).                                             |
|                                     |                                                      |
|                                     | - Understands symmetry, regularity, and              |
|                                     |   distinguishes between 2D and 3D shapes.            |
|                                     |                                                      |
|                                     | - Identifies and names 3D figures (rectangular       |
|                                     |   prisms, cubes), counting vertices, edges, and      |
|                                     |   faces.                                             |
|                                     |                                                      |
|                                     | - Understands shape movements: slides, flips, and    |
|                                     |   turns; distinguishes symmetry and asymmetry.       |
|                                     |                                                      |
|                                     | - Follows and gives directions using a coordinate    |
|                                     |   plane.                                             |

|                                     |   personal well-being and to their relationships     |
