| Español                            | Expresión oral y vocabulario                          |
|                                    |                                                       |
|                                    | - Prepara una obra teatral a partir de un texto       |
|                                    |   narrativo, con todas sus características (actos,    |
|                                    |   escenas, acotaciones, personajes, etc.)             |
|                                    |                                                       |
|                                    | - Presenta ante una comunidad escolar las diferencias |
|                                    |   y semejanzas de los principales textos narrativos   |
|                                    |   (novela, cuento, fábula, etc.)                      |
|                                    |                                                       |
|                                    | Lectura                                               |
|                                    |                                                       |
|                                    | - Reconoce las características gramaticales y la      |
|                                    |   redacción de una carta, así como el vocabulario     |
|                                    |   formal adecuado al tipo de documento.               |
|                                    |                                                       |
|                                    | - Reflexiona sobre las diferentes figuras literarias: |
|                                    |   metáfora, símil, hipérbole, onomatopeyas y          |
|                                    |   aliteración, haciendo énfasis en la importancia del |
|                                    |   lenguaje figurado como recurso para llamar la       |
|                                    |   atención del lector.                                |
|                                    |                                                       |
|                                    | - Conoce las características y las diferentes formas  |
|                                    |   de esquematizar un texto leído: mapa mental, mapa   |
|                                    |   conceptual, diagrama de flujo y cuadro sinóptico.   |
|                                    |                                                       |
|                                    | Escritura                                             |
|                                    |                                                       |
|                                    | - Escribe una carta formal en la que plantea un       |
|                                    |   asunto particular, explorando diferentes modelos y  |
|                                    |   estilos de cartas formales.                         |
|                                    |                                                       |
|                                    | - Escribe y analiza poemas describiendo el mundo      |
|                                    |   social en que se presenta la historia o anécdota    |
|                                    |   del mismo.                                          |
|                                    |                                                       |
|                                    | Reflexión de la lengua gramática y ortografía         |
|                                    |                                                       |
|                                    | - Analiza y valora la finalidad y características de  |
|                                    |   los textos reglamentarios en cuanto a su            |
|                                    |   estructura, haciendo énfasis en los verbos y        |
|                                    |   tiempos verbales.                                   |
|                                    |                                                       |
|                                    | - Conoce y diferencia los tipos de sustantivos en un  |
|                                    |   texto: comunes, propios, concretos, abstractos,     |
|                                    |   individuales, colectivos, contables e incontables,  |
|                                    |   primitivos, derivados, simples y compuestos.        |
|                                    |                                                       |
|                                    | - [Utiliza correctamente el acento diacrítico en      |
|                                    |   palabras monosílabas (tú/tu, él/el, sé/se,          |
|                                    |   etc.).]{.mark}                                      |
|                                    |                                                       |
|                                    | - [Aplica reglas ortográficas para la realización de  |
|                                    |   la división silábica, clasificándolas correctamente |
|                                    |   en agudas, graves, esdrújulas y                     |
|                                    |   sobreesdrújulas.]{.mark}                            |
+------------------------------------+-------------------------------------------------------+
| Math                               | Multi-step Equations                                  |
|                                    |                                                       |
|                                    | - Solves two-step equations involving integers,       |
|                                    |   decimals, and fractions using inverse operations.   |
|                                    |                                                       |
|                                    | - Solves multi-step equations by applying properties  |
|                                    |   of operations, such as the distributive property    |
|                                    |   and combining like terms, to isolate the variable.  |
|                                    |                                                       |
|                                    | - Writes and solves equations to represent real-world |
|                                    |   and mathematical problems.                          |
|                                    |                                                       |
|                                    | Linear Functions                                      |
|                                    |                                                       |
|                                    | - Uses tables and graphs to determine if a relation   |
|                                    |   is a function, identifies proportional              |
|                                    |   relationships, and explains the unit rate in a      |
|                                    |   proportional graph.                                 |
|                                    |                                                       |
|                                    | - Graphs linear functions from tables of values and   |
|                                    |   interprets the graph by identifying the slope (m)   |
|                                    |   and y-intercept (b), and describes the relationship |
|                                    |   between the variables.                              |
|                                    |                                                       |
|                                    | - Translates real-world situations into linear        |
|                                    |   functions and solves related word problems, using   |
|                                    |   the equation y = mx + b and understanding its       |
|                                    |   components.                                         |
|                                    |                                                       |
|                                    | Ratios and Proportions                                |
|                                    |                                                       |
|                                    | - Calculates unit rates in various contexts (length,  |
|                                    |   area, and other real-world scenarios) and           |
|                                    |   represents them using tables and graphs.            |
|                                    |                                                       |
|                                    | - Determines whether given ratios form a proportion,  |
|                                    |   identifies the constant of proportionality, and     |
|                                    |   uses graphs to represent a proportion.              |
|                                    |                                                       |
|                                    | - Applies proportional reasoning to solve multi-step  |
|                                    |   problems, including real-world applications such as |
|                                    |   scaling images, maps, and models.                   |
|                                    |                                                       |
|                                    | - Uses scale factors to analyze the enlargement and   |
|                                    |   reduction of figures, particularly triangles, and   |
|                                    |   applies proportional reasoning to solve geometric   |
|                                    |   problems.                                           |
|                                    |                                                       |
|                                    | Percentages                                           |
|                                    |                                                       |
|                                    | - Computes percentages of a given number, determines  |
|                                    |   what percentage one number is of another, and       |
|                                    |   solves for an unknown in percentage equations.      |
|                                    |                                                       |
|                                    | - Solves word problems involving percentage increase, |
|                                    |   percentage decrease, and absolute change using      |
|                                    |   proportional relationships.\                        |
|                                    |   Understands and applies financial concepts, such as |
|                                    |   interest, capital, tax, and discounts, by solving   |
|                                    |   real-world word problems involving percentage       |
|                                    |   calculations.                                       |
|                                    |                                                       |
|                                    | - Calculates discounts, determines final prices after |
|                                    |   applying discounts and taxes, and analyzes how      |
|                                    |   percentages impact real-world financial decisions.  |
|                                    |                                                       |
|                                    | Probability and Statistics                            |
|                                    |                                                       |
|                                    | - Finds the mean, median, mode, and range of a data   |
|                                    |   set, and uses these measures of central tendency    |
|                                    |   and spread to compare two sets of data. Understands |
|                                    |   the difference between a population and a sample.   |
|                                    |                                                       |
|                                    | - Creates and interprets scatter plots and            |
|                                    |   histograms, and determines which type of graph is   |
|                                    |   most appropriate for representing different types   |
|                                    |   of data.                                            |
|                                    |                                                       |
|                                    | - Calculates experimental probabilities, predicts     |
|                                    |   outcomes based on data, and finds the probability   |
|                                    |   of a complementary event.                           |
|                                    |                                                       |
|                                    | - Uses organized methods such as tree diagrams to     |
|                                    |   list possible outcomes, finds the sample space, and |
|                                    |   calculates the probability of an event.             |
+------------------------------------+-------------------------------------------------------+
| English                            | Inverted Subjects and Verbs                           |
|                                    |                                                       |
|                                    | - Practices how to make these sentences:              |
|                                    |                                                       |
|                                    |   - With question words such as what, when, where,    |
|                                    |     why, and how.                                     |
|                                    |                                                       |
|                                    |   - After some place expressions.                     |
|                                    |                                                       |
|                                    |   - After negative expressions.                       |
|                                    |                                                       |
|                                    |   - In some conditionals.                             |
|                                    |                                                       |
|                                    |   - After some comparisons.                           |
|                                    |                                                       |
|                                    | Pronouns                                              |
|                                    |                                                       |
|                                    | - Distinguishes between subjective and objective      |
|                                    |   pronouns.                                           |
|                                    |                                                       |
|                                    | - Avoids common mistakes when using them.             |
|                                    |                                                       |
|                                    | - Uses personal pronouns in place of noun phrases.    |
|                                    |                                                       |
|                                    | - Uses them to refer back to people and things that   |
|                                    |   have already been identified.                       |
|                                    |                                                       |
|                                    | - Uses them to show a person and a number.            |
|                                    |                                                       |
|                                    | - Writes sentences where they act as the subject of a |
|                                    |   clause.                                             |
|                                    |                                                       |
|                                    | Adjectives and Adverbs                                |
|                                    |                                                       |
|                                    | - Understands that an adjective describes a noun or   |
|                                    |   pronoun.                                            |
|                                    |                                                       |
|                                    | - Recognizes and recalls adjectives as degrees of     |
|                                    |   comparison.                                         |
|                                    |                                                       |
|                                    | - Understands that an adverb describes a verb or      |
|                                    |   anything apart from a noun or pronoun.              |
|                                    |                                                       |
|                                    | Prepositions, Conjunctions, and Interjections         |
|                                    |                                                       |
|                                    | - Identifies that prepositions and conjunctions are   |
|                                    |   the connecting elements in sentences.               |
|                                    |                                                       |
|                                    | - Finds the link between words as the key to          |
|                                    |   identifying prepositions.                           |
|                                    |                                                       |
|                                    | - Practices sentences and recognizes the two          |
|                                    |   guidelines about using prepositions: avoids         |
|                                    |   unnecessary prepositions, and avoids ending         |
|                                    |   sentences with prepositions.                        |
|                                    |                                                       |
|                                    | - Defines the term conjunction.                       |
|                                    |                                                       |
|                                    | - Identifies different types of conjunctions.         |
|                                    |                                                       |
|                                    | - Lists conjunctions within each category.            |
|                                    |                                                       |
|                                    | - Explains the purpose of conjunctions.               |
|                                    |                                                       |
|                                    | - Identifies an interjection.                         |
|                                    |                                                       |
|                                    | - Understands how to use an interjection properly.    |
|                                    |                                                       |
|                                    | - Knows several common interjections and their        |
|                                    |   typical uses.                                       |
|                                    |                                                       |
|                                    | Clauses and Complex Sentences                         |
|                                    |                                                       |
|                                    | - Understands that a clause takes the place of the    |
|                                    |   direct object of a verb.                            |
|                                    |                                                       |
|                                    | - Recognizes the different types of clauses.          |
|                                    |                                                       |
|                                    | - Understands that a complex sentence contains an     |
|                                    |   independent clause with one or more subordinate     |
|                                    |   clauses.                                            |
