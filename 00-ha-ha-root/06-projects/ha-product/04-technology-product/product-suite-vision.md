### Example Integration Flows

**Flow 1: New Project Alignment**
1. New project created in Portfolio Dashboard
2. Legacy Contribution Score auto-calculated
3. Skills Gap Analyzer identifies training needs
4. Course Maker suggests onboarding for team
5. Market Radar provides competitive context

**Flow 2: Project Retrospective**
1. Project marked complete in Portfolio
2. Lesson Extractor analyzes all project data
3. Knowledge Graph updated with new learnings
4. Team Health Dashboard updated with feedback
5. Legacy Contribution Score finalized

**Flow 3: Strategic Planning**
1. Scenario Planner models 3 futures
2. Legacy Builder checks alignment with each
3. Portfolio Dashboard simulates resource needs
4. Skills Gap Analyzer projects future capabilities needed
5. Course Maker pre-builds training for likely scenarios
